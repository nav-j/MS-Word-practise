## **MS Word Practice Task – Level: Beginner to Intermediate**

### Task Title: *“My Digital Skills Report”*

You have to create a 2–3 page document in MS Word using the instructions below.

---

### PART 1: Home Tab (Formatting & Text Tools)

1. Type the title at the top of the page:
   **MY DIGITAL SKILLS REPORT**

2. Apply the following:

   * Font: *Arial*
   * Font Size: *18*
   * Make it **Bold**
   * Align it to the **Center**

3. Below the title, type this paragraph:

   > In today’s world, digital skills are essential for learning, working, and communication. MS Word is a powerful word processing tool that helps users create professional documents easily.

4. Format the paragraph:

   * Font size: *12*
   * Justify the text
   * Change the line spacing to *1.5*

5. Now create a **bulleted list** of at least 5 digital skills:

   * Typing
   * Internet browsing
   * Email communication
   * Online safety
   * Microsoft Office

6. Change the bullet style and make the list **Blue color**.

---

### PART 2: Insert Tab (Objects & Media)

7. Insert a **picture** related to technology or computers.

8. Insert a **table** (3 rows × 2 columns) and fill it like this:

| Skill    | Level     |
| -------- | --------- |
| Typing   | Good      |
| MS Word  | Very Good |
| Internet | Excellent |

9. Insert **page number** at the bottom center.

10. Insert a **SmartArt** that shows 3 learning levels:

* Beginner
* Intermediate
* Advanced

---

### PART 3: Design Tab (Themes & Styles)

11. Apply any **Word Theme** of your choice.

12. Change the **Document Colors and Fonts**.

13. Add a **Watermark** that says:
    **PRACTICE COPY**

---

### PART 4: Layout Tab (Page Setup)

14. Set margins to **Narrow**.

15. Change the **Page Orientation** to **Portrait**.

16. Set **Spacing After** = *6 pt* for all paragraphs.

17. Add a **Page Border** (simple box style is fine).

---

### Final Result Should Have

✔ Formatted title
✔ Paragraph with spacing & justification
✔ Colored bullet list
✔ Inserted image
✔ Table & SmartArt
✔ Page numbers
✔ Theme, watermark, border
✔ Correct margins & layout

Here are the **complete, step-by-step solution instructions** for your **MS Word task** using the **Home, Insert, Design and Layout tabs**.

---

## ✅ SOLUTION: “My Digital Skills Report” – MS Word

---

## PART 1 – HOME TAB (Formatting & Text)

### 1. Type the Title

1. Open **MS Word**
2. At the top type:
   **MY DIGITAL SKILLS REPORT**

### 2. Format the Title

1. Select the title text
2. Go to **Home tab**
3. Set the following:

   * Font: **Arial**
   * Font Size: **18**
   * Click **Bold (B)**
   * Click **Center alignment**

---

### 3. Type the Paragraph

Below the title, type:

> In today’s world, digital skills are essential for learning, working, and communication. MS Word is a powerful word processing tool that helps users create professional documents easily.

---

### 4. Format the Paragraph

1. Select the paragraph
2. Go to **Home tab**
3. Set:

   * Font size: **12**
   * Click **Justify alignment**
   * Click **Line & Paragraph Spacing**
   * Select **1.5**

---

### 5. Create Bullet List

Below the paragraph, write:

Typing
Internet browsing
Email communication
Online safety
Microsoft Office

Now:

1. Select the list
2. Go to **Home tab**
3. Click **Bullets**
4. Click arrow next to bullets and choose **different style**
5. Change **font color to Blue**

---

## PART 2 – INSERT TAB (Objects & Media)

### 6. Insert Image

1. Go to **Insert tab**
2. Click **Pictures** → **This Device**
3. Select any **technology-related image**
4. Click **Insert**
5. Resize and move as needed

---

### 7. Insert Table (3 x 2)

1. Go to **Insert tab**
2. Click **Table**
3. Select **3 rows × 2 columns**
4. Enter this data:

| Skill    | Level     |
| -------- | --------- |
| Typing   | Good      |
| MS Word  | Very Good |
| Internet | Excellent |

---

### 8. Insert Page Number

1. Go to **Insert tab**
2. Click **Page Number**
3. Choose **Bottom of Page**
4. Select **Center**

---

### 9. Insert SmartArt

1. Go to **Insert tab**
2. Click **SmartArt**
3. Choose **Process** or **List**
4. Add 3 items:

   * Beginner
   * Intermediate
   * Advanced

---

## PART 3 – DESIGN TAB (Theme & Background)

### 10. Apply Theme

1. Go to **Design tab**
2. Click **Themes**
3. Choose any theme you like

---

### 11. Change Document Colors and Fonts

1. In **Design tab**, click **Colors** → choose a color set
2. Click **Fonts** → choose a font style

---

### 12. Add Watermark

1. Go to **Design tab**
2. Click **Watermark**
3. Choose **Custom Watermark**
4. Select **Text watermark**
5. Type: **PRACTICE COPY**
6. Click **OK**

---

## PART 4 – LAYOUT TAB (Page Setup)

### 13. Change Margins

1. Go to **Layout tab**
2. Click **Margins**
3. Click **Narrow**

---

### 14. Set Orientation

1. Still in **Layout tab**
2. Click **Orientation**
3. Select **Portrait**

---

### 15. Set Spacing After

1. Press **Ctrl + A** (Select all)
2. Go to **Layout tab**
3. In **Spacing (After)** write **6 pt**

---

### 16. Add Page Border

1. Go to **Design tab**
2. Click **Page Borders**
3. Choose **Box style**
4. Click **OK**

---
